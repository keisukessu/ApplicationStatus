# ApplicationStatus - 就活応募状況管理アプリ

就活における企業への応募状況をコンソール上で管理できるCLIアプリケーションです。
企業の追加、ステータス管理、内定率の確認が簡単に行えます。


## 機能

- **企業追加** - 応募先の企業を登録する（初期ステータス: ENTRY）
- **一覧表示** - 登録済みの全企業とステータスを一覧表示する
- **ステータス変更** - 企業の応募ステータスを更新する
- **内定率表示** - 全企業に対する内定（OFFER）の割合を表示する
- **データ保存** - 企業データはCSVファイルに自動保存される

## ステータス一覧

| ステータス | 意味 |
|---|---|
| ENTRY | エントリー済み（応募段階） |
| INTERVIEW | 面接中 |
| OFFER | 内定 |
| REJECTED | 不採用 |

## ファイル構成

```
ApplicationStatus/
├── Main.java        # メイン
├── UI.java          # ユーザーインターフェース（メニュー表示・入力処理）
├── Service.java     # ロジック（追加・更新・内定率計算）
├── FileService.java # データ処理（CSV読み込み・保存）
├── Company.java     # 企業クラス
├── Status.java      # 応募ステータスのenum
└── companies.csv    # データの保存先
```

## 実行方法

```bash
# コンパイル
javac Main.java
#文字化けする場合代わりに下記のコード
javac -encoding UTF-8 *.java

# 実行
java Main
```
## 使い方

起動するとメニューが表示されます。番号を入力して操作を選択してください。

```
1: 企業追加
2: 一覧表示
3: ステータス変更
4: 内定率表示
0: 終了
```
